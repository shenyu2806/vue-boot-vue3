# 项目介绍

<h1>团星网后台管理系统</h1>
<p>Vue3 +  vite  + element-plus + pinia + icons</p>
<p>支持二开，商业使用</p>

# 安装命令可选 npm/cnpm 
# 安装cnpm
npm install cnpm
# vue安装router
cnpm install vue-router@4
# 启动项目
## 安装依赖
cnpm install
## 启动
npm run dev
## 默认账号
